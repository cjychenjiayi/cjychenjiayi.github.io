# Jiayi Chen · Academic homepage

This site is based on the open-source [AcadHomepage](https://github.com/RayeRen/acad-homepage.github.io) GitHub Pages template and keeps its Jekyll, Markdown, and responsive layout structure.

Personal content has been replaced with Jiayi Chen's research, publications, CV links, and selected MP4 research previews. No PPT/PPTX files, paper source archives, or project folders are included.

For local preview, install the template's Jekyll dependencies and run the included run_server.sh script. GitHub Pages can build the repository directly from the main branch.
